# OpenDesk Pages

[![Deploy to GitHub Pages](https://github.com/openharmony/opendesk-pages/actions/workflows/pages.yml/badge.svg)](https://github.com/openharmony/opendesk-pages/actions/workflows/pages.yml)

> 🚀 OpenHarmony 社区孵化的通用智能体项目页面

## 📖 项目简介

OpenDesk Pages 是 OpenHarmony 社区孵化的通用智能体项目的官方网站页面。本项目展示了 OpenDesk 的核心功能和愿景，为用户提供了解和接触 OpenDesk 的入口。

## 🌟 特性

- ✨ 现代化的响应式设计
- 🎨 优雅的渐变色 UI
- ⚡ 轻量级静态页面，加载快速
- 📱 完全响应式，支持各种设备
- 🔒 使用 GitHub Pages 安全托管

## 🚀 快速开始

### 本地预览

1. 克隆仓库：
```bash
git clone https://github.com/openharmony/opendesk-pages.git
cd opendesk-pages
```

2. 打开 `index.html` 文件即可在浏览器中预览

### 部署到 GitHub Pages

本项目已配置 GitHub Actions 自动部署工作流，推送到 `main` 分支后会自动构建和部署。

#### 手动部署步骤：

1. 在 GitHub 仓库设置中启用 GitHub Pages
2. 选择 Source 为 "GitHub Actions"
3. 推送代码到 `main` 分支，或手动触发工作流

## 📂 项目结构

```
opendesk-pages/
```
