参与 OpenDesk 的开发工作，并不意味着需要帮 OpenDesk 写代码。我们也许已经经过（也许正在经过）那个代码才是核心贡献的年代。截止到 2026年6月，OpenDesk 的代码量达到约 20万行，其中累计已有约 60% 的代码完全由 Agent 开发，其中相当一部分由 OpenDesk 自己开发。6月份新增代码约有 85-90% 完全由Agent开发 [参考标准](#正确标记AI开发的代码)。

## 找到 OpenDesk 的官方代码仓库

- OpenDesk主仓库：[https://gitcode.com/openharmony-robot/OpenDesk](https://gitcode.com/openharmony-robot/OpenDesk)
- OpenDesk 鸿蒙Electron适配: [https://gitcode.com/openharmony-robot/OpenDesk-ohos](https://gitcode.com/openharmony-robot/OpenDesk-ohos)

## 有哪些工作可以参与？

- **作为使用者**，你可以将你想要的功能通过。或者你发现的bug 以 issue 的形式提在 [这里](https://gitcode.com/openharmony-robot/OpenDesk/issues)。开发团队会定期浏览 issue 页面，通常在工作日内在24小时内会给出回复；
