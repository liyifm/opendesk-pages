## 1. 安装 OpenDesk Desktop

OpenDesk Desktop 提供图形化界面，无需安装 NodeJS 等运行环境。

### 1.1 下载安装包

从 [Release Notes](/news) 页面获取对应平台的最新安装包：

| 平台 | 安装方式 |
|------|----------|
| **macOS** | 下载 `.dmg` 安装包，双击拖入 Applications |
| **Windows** | 下载 `.exe` 安装包，运行安装向导 |
| **HarmonyOS PC** | 在应用市场搜索 OpenDesk 安装 |

> 安装包和平台差异说明持续更新中，请关注 [Release Notes](/news) 获取最新版本。

### 1.2 首次启动

安装完成后双击 OpenDesk 图标启动应用。

---

## 2. 快速完成初始配置

首次启动时，如果尚未配置模型，会弹出提示：

<img src="/assets/images/quickstart/firsttime.png" alt="首次配置提示" />

在输入框中键入 `/config` 回车，打开模型配置面板，按提示依次配置：

1. **模型服务商** — 选择 OpenAI、硅基流动、智谱等（OpenDesk 支持所有提供 OpenAI 标准模型服务的厂商）
2. **模型别名** — 为所选模型起一个简短别名
3. **默认模型** — 设为日常使用的默认模型

配置完成后即可开始对话。

> 💡 大部分配置都可以通过自然语言对话完成，无需手动操作设置界面。直接告诉 AI "帮我配置 xxx 模型" 即可。

### 2.1 与朋友分享配置

输入 `/quickpass config base` 导出配置口令，朋友在聊天框粘贴口令即可一键应用。

<img src="/assets/images/quickstart/quickpass.jpg" alt="配置口令导出" />

## 3. 界面概览

启动后你会看到三栏布局：

| 区域 | 功能 |
|------|------|
| **侧边栏** | 任务列表，支持搜索，底部为设置入口 |
| **对话区** | 核心交互区：任务标题栏 + 消息列表 + 输入框 |
| **结果区** | 右侧面板：展示生成的文件、执行步骤、文件变更；可展开/收起，也可导出为文件 |

---

## 4. 创建你的第一个任务

用一句话描述需求即可。试试这样对它说：

- 「查看我今天的日程安排」
- 「帮我把这份销售数据做成分析报告」
- 「读一下这个项目的代码，帮我理清架构」

<img src="/assets/images/manual/quickstart_task.gif" alt="创建第一个任务" />

### 选择工作目录（可选）

1. 点击输入框右下角的「选择文件夹」按钮
2. 选择当前任务要使用的目录
3. OpenDesk 仅在该目录内操作文件，不会影响其他位置

### 补充上下文（可选）

| 方式 | 说明 |
|------|------|
| `@` 引用 | 引用文件、文档、规则等上下文 |
| 粘贴截图 | `Ctrl/Cmd + V` 粘贴剪贴板中的图片 |
| 上传文件 | 拖拽文件到输入框 |
| 补充说明 | 明确目标、范围、约束和预期输出 |

## 5. 查看结果与追问

任务完成后，右侧结果区包含三类视图：

| 视图 | 内容 |
|------|------|
| **产物视图** | 任务生成的文件（文档、数据、图片等） |
| **变更视图** | 文件的新增/删除/修改（diff 高亮对比） |
| **记忆视图** | 任务过程中自动维护的结构化记忆 |

追问无需重复背景，直接说改什么即可：

- 「再加一个饼图展示占比」
- 「导出为 PDF」

## 6. 进阶配置

以下配置均通过对话完成，与 CLI 共享同一套智能体能力。

### 6.1 配置消息渠道

让 AI 7×24 小时在线，支持微信、飞书等渠道接入：

- [飞书一键扫码配置](/channels/feishu)
- [微信一键扫码配置](/channels/weixin)

### 6.2 配置搜索引擎

默认支持 Bing、DuckDuckGo、ArXiv。也可接入 Exa、Tavily、Brave 等付费引擎，直接告诉 AI "配置一下 Exa 搜索" 即可。

<img src="/assets/images/quickstart/searchconfig.gif" alt="搜索引擎配置" />

### 6.3 安装 MCP Server / 技能

一句话安装：直接告诉 AI "帮我安装 xxx MCP Server" 或 "帮我安装 xxx 技能"，OpenDesk 自动搜索配置。

<img src="/assets/images/quickstart/mcpinstall.gif" alt="MCP Server 安装" />
